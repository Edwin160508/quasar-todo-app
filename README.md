# Quasar App

> WIP
# Primeiro Passo Instalar cli Quasar
1- Abrir o terminal 
    npm install -g quasar-cli
2- Baixar o projeto 
    npm install
3- Rodar o projeto
    quasar dev
# Segundo passo Instalar cordova    
npm install -g cordova

# Opcional
## Caso queira que sua app seja para IOS
   quasar dev -m cordova -T ios
